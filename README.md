# GTA Casino Solver

##### Because remembering fingerprints was more annoying than writing a script xd


Remembering fingerprint patterns gets annoying pretty fast — this script exists purely to save that effort. Nothing fancy, just a convenience thing I made out of curiosity.

## What it does

- Helps solve the casino fingerprint puzzle
- Removes the need to memorise fingerprint patterns
- Runs locally as an AutoHotkey script

That’s all it’s meant to do.

## Why I made this

Mostly for fun, and because manually remembering patterns felt unnecessary.  
I wanted to see if I could make the process a little smoother, so I did.

## Notes

- Built using AutoHotkey
- The file name is intentionally kept fixed
- Not a full mod menu or anything like that
- No active development planned

Shared as-is, mostly for reference and curiosity.
## Credits

This script is based on an earlier fingerprint recognition tool by an unknown original author, which already handled pattern detection and tooltips.

I extended it by adding:
- an automatic fingerprint selection algorithm
- a toggle between manual and auto solving modes

Shared with respect to the original work.
